# Question Pack: Handoff Continuation

Use when a grilling session is too long, branches into a prototype, crosses context windows, or must be resumed by another agent.

| ID | Question | Purpose | Evidence to inspect first | Recommended answer pattern | Blocking decision |
|---|---|---|---|---|---|
| HC-01 | What is the destination the next session must preserve? | Keep continuity. | current plan, decisions | State the destination in one sentence. | destination |
| HC-02 | Which decisions are already confirmed? | Avoid re-litigating. | session notes, report | List confirmed decisions with IDs. | decision carryover |
| HC-03 | Which questions remain open and why? | Preserve blockers. | open items | List P0/P1 open questions and owners. | open blockers |
| HC-04 | Which evidence should the next agent read first? | Reduce context load. | files, reports, code paths | Provide ordered links or paths. | evidence order |
| HC-05 | Which skills should the next session invoke? | Make continuation actionable. | downstream map | Recommend skill sequence. | next skills |
| HC-06 | What must the next agent avoid doing? | Preserve safety. | risk gates, non-goals | List forbidden actions. | safety boundary |
| HC-07 | What is the next single question to ask the user? | Resume grilling cleanly. | open blockers | Provide one question. | resume point |
